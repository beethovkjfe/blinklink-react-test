## Blinklink Code test

Frontend codebase for blinklink code test

## Requirements

- Node `v16.17.0`
- NPM `v8.15.0`

### Setup Instructions:

`git clone git@github.com:beethovkjfe/blinklink-react-test.git`

`cd blinklink-react-test`

`npm install --legacy-peer-deps`

`npm run reset`

### Run Instructions:

`npm start`

Then go to 

`localhost:3000`

### Test

`npm run test`

### Lint

`npm run lint`
